# 以太魔导科技：装置与元件设定集 (Aether Magitech: Devices & Mechanics)

本文档记录了基于“以太魔导科技”世界观的背景设定、装置设计以及建议加入游戏的新元件机制。

## 1. 世界观概要 (Worldview Overview)

**以太 (Aether/Ether)**：一种无处不在的高能流体，存在于空间的每一个角落。它具有类似液体的流动性，但又遵循电磁场的某些规律。
- **蓝色态**：稳定的环境以太。
- **绿色态**：被抽取并在导线/管道中流动的可用能量。
- **黄色/红色态**：高压、过载、即将失控的危险能量。

**魔导器 (Magitech Artifacts)**：利用以太能量驱动的机械装置。它们通常由黄铜、水晶和经过蚀刻的导魔金属制成。核心原理是将以太能量转化为物理现象（光、热、动能、力场）。

## 2. 生活与民用装置 (Civilian Devices)

这些装置展示了以太科技如何渗透到日常生活中，体现“魔法即科技”的主题。

### **以太辉光灯 (Aether-Glow Lamp)**
*   **原理**：利用低压以太流激发充气玻璃管中的稀有气体。
*   **外观**：复古的黄铜底座，上方是充满螺旋状灯丝的玻璃罩。
*   **机制灵感**：作为一种消耗极低的负载，用于点亮地图或指示电路通畅。

### **恒温力场箱 (Stasis Box / "Magic Fridge")**
*   **原理**：通过持续消耗能量维持一个局部的“时间/熵减缓”力场，从而无限期保存食物。
*   **副作用**：如果能量中断，力场崩溃会瞬间释放累积的热量，导致食物瞬间腐烂甚至燃烧。
*   **机制灵感**：需要持续供电的“饥饿”元件，一旦断电就会导致周围过热（惩罚机制）。

### **自动清洁傀儡 (Autoclean Golem)**
*   **原理**：小型发条驱动的机器人，核心是一块微型以太电池。它们会吸附灰尘并将其转化为微量的以太补充自身。
*   **外观**：像是一只长着扫帚尾巴的金属螃蟹。

## 3. 军用与重工装置 (Military & Industry)

这些大型装置展示了以太科技暴力和宏大的一面。

### **以太悬浮列车 (Maglev Aether-Train)**
*   **原理**：铁轨本身就是巨大的能量传输线路。列车通过底部的反重力发生器消耗巨额能量悬浮，并利用磁轨加速。
*   **外观**：流线型的装甲车头，车身布满散热格栅，高速行驶时周围会产生蓝色的电离空气。

### **城防护盾发生器 (City Shield Generator)**
*   **原理**：将巨量以太能量泵入高空的节点，形成一个巨大的半球形斥力场。
*   **弱点**：能耗极高，需要全城的发电厂满负荷运作。过载时会产生彩虹色的极光，被称为“女神的叹息”。

### **过载榴弹 (Overload Grenade)**
*   **原理**：一个不稳定的电容，设计目的就是为了“故障”。拉开引信破坏稳压阀，3秒后内部能量突破临界值引发爆炸。
*   **机制灵感**：不仅是武器，也可以作为炸开地形障碍的工具。

### **动能冲击钻 (Kinetic Drill)**
*   **原理**：利用“Maker”产生的实体粒子，经过极短距离的超高压加速，反复撞击岩层。
*   **应用**：矿业开采，或作为攻城武器破坏城墙。

## 4. 游戏内元件提案 (Proposed Game Mechanics)

基于上述设定，以下是可以加入到游戏中的具体元件设计，旨在丰富目前的“生产-传输-消耗”循环。

### **A. 逻辑与控制类**

1.  **分流器 (Splitter / Prism)**
    *   **描述**：将输入的能量流平均分配到三个出口。
    *   **用途**：解决单根导线承载能力有限的问题，将高压流分散为低压流。
    *   **视觉**：一个透明的三角形水晶棱镜。

2.  **开关/闸门 (Switch / Valve)**
    *   **描述**：可以通过点击手动开启或关闭的导线。
    *   **用途**：在过载紧急时刻切断电路，或者用于切换供能路线。
    *   **视觉**：一个带有把手的巨大黄铜闸刀。

3.  **二极管/单向阀 (Diode / Check Valve)**
    *   **描述**：只允许能量单向流动，防止回流。
    *   **用途**：构建复杂的循环电路，防止反向波涌导致源头爆炸。
    *   **视觉**：带有箭头标识的管道段。

### **B. 增强与处理类**

4.  **增压泵/变压器 (Amplifier / Transformer)**
    *   **描述**：消耗周围的环境能量（类似抽取泵），将通过它的能量流“加压”（增加数值），但会产生大量热量（需要散热）。
    *   **用途**：长距离传输导致能量衰减（如果有此机制）时的中继站，或者强行提升能量供给给高耗能设备。

5.  **散热器 (Heatsink / Radiator)**
    *   **描述**：本身不传输能量，但可以放置在其他元件旁边。它会吸收邻近元件的“过载值”或“热量”，并将其排放到空气中。
    *   **用途**：允许周围的导线或机器在超负荷状态下工作更长时间而不爆炸。
    *   **视觉**：密集的金属鳍片，工作时会发红。

### **C. 高级粒子交互**

6.  **护盾发生器 (Shield Generator)**
    *   **描述**：消耗大量能量，在周围一圈形成保护力场。
    *   **用途**：保护内部的元件免受外部爆炸的波及，或者阻挡（反弹）错误的粒子。

7.  **炼金转化器 (Alchemical Transmuter)**
    *   **描述**：这是一种高级的“Maker”。它不凭空产生粒子，而是接收一种基础粒子，消耗能量将其“转化”为高级粒子（比如金粒子、以太结晶）。
    *   **用途**：增加生产链的复杂度。 基础粒子 -> 加速 -> 转化器 -> 高级粒子 -> 终点。
